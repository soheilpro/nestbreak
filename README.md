nestbreak
========
A Twitter API proxy in Node.js.

Requirements
----------
[Node.js](http://nodejs.org)

Status
-------
Not working now.

How to use?
----------
1. Run the server:

    node nestbreak

2. Set your Twitter client's API Root URL to:

    http://<i>server</i>:8080
